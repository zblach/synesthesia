synesthesia
=========

colorized pipe for visual pattern analysis.
# requirements
* ansicolors module

# usage 

## example 

    xxd <options> [filename] | synesthesia.py
    hexdump [filename] | synesthesia.py

# features 
### pipes supported:
* xxd (binary & hex)
* hexdump
  
### color schemes supported:
* Default
* UglyRainbowHex
* NicerRainbowHex
* BracketCounter
* Composite
